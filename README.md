## Design Direction

The visual language draws from editorial print — no cards, no thumbnails. Projects are presented as an index: thin rules, large serif titles, minimal metadata. The aesthetic prioritises negative space over decoration.

## Running Locally

No build step required. Open directly in a browser:

```bash
open index.html
```

Or drag `index.html` into any browser window.
